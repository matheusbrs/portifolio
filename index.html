Calculadora de IMC

Meu projeto é uma calculadora de Índice de Massa Corporal (IMC) que permite aos usuários calcular sua proporção de peso para altura. O IMC é uma medida comumente utilizada para avaliar se uma pessoa está dentro de uma faixa de peso saudável com base na sua altura. Esta ferramenta é valiosa para indivíduos que desejam monitorar sua saúde e adotar hábitos de vida mais saudáve
